# ImageMaster Web (Safari, iPhone 14 Pro)

Веб-приложение для смены формата изображений. Адаптировано под запуск в **Safari на iPhone 14 Pro**: безопасные отступы (safe area), удобные размеры кнопок, тёмная тема.

## Возможности

- **Формат:** Квадрат (1:1), Вертикальный (3:4), Сторис (9:16), Свой формат (два поля).
- **Фон:** белый, чёрный или размытый (уменьшение/увеличение картинки для эффекта blur).
- **Выбор фото:** кнопка «Выбрать фото», несколько файлов.
- **Превью:** по выбранному изображению при смене формата и фона.
- **Старт:** обработка всех выбранных фото. На iPhone (Safari) открывается меню «Поделиться» — выберите **«Сохранить изображение»**, чтобы сохранить в приложение «Фото». На десктопе результат скачивается в папку «Загрузки».

Логика совпадает с десктопной версией: холст по выбранному ratio, изображение по центру, без обрезки.

## Запуск

### С телефона (iPhone 14 Pro)

1. Разместите папку `web` на любом веб-сервере (или используйте хостинг вроде GitHub Pages, Netlify).
2. Откройте в **Safari** URL приложения (например `https://ваш-сайт.ru/web/`).
3. При желании: **Добавить на экран «Домой»** — тогда приложение откроется в полноэкранном режиме без панели Safari.

### Локально с Mac

```bash
cd web
python3 -m http.server 8080
```

Откройте в браузере: `http://localhost:8080`. Для проверки на телефоне в той же Wi‑Fi сети: `http://IP-вашего-Mac:8080`.

### Локально без сервера

Можно открыть `index.html` напрямую (двойной клик). На iOS из-за ограничений CORS и доступа к файлам часть сценариев может вести себя иначе — предпочтительно запускать через сервер или с хоста.

## Адаптация под iPhone 14 Pro

- `viewport-fit=cover` и `env(safe-area-inset-*)` для отступов от выреза и индикатора дома.
- Мета-теги `apple-mobile-web-app-capable`, `theme-color` для режима «Добавить на экран «Домой»».
- Минимальная высота интерактивных элементов 44px.
- Максимальная ширина контента 430px для удобного чтения на узком экране.
- Тёмная тема и контрастные акценты (бирюзовый).

## Сохранение в «Фото» на iPhone

После нажатия «Старт» для каждого обработанного изображения открывается меню **Поделиться**. Выберите **«Сохранить изображение»** — фото попадёт в приложение «Фото» (Фотоплёнка). Если меню закрыть без сохранения, будет предложено скачивание файла. На компьютере результат по-прежнему скачивается в папку «Загрузки».
